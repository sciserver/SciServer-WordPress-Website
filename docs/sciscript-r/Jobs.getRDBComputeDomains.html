Jobs.getRDBComputeDomains {SciServer}R Documentation

Get database compute domains

Description

Gets a list of all registered Relational Database (RDB) compute domains that the user has access to.

Usage

Jobs.getRDBComputeDomains()

Value

a list containing the definition of RDB compute domains.
