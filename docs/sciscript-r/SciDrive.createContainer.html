SciDrive.createContainer {SciServer}R Documentation

Create container.

Description

Creates a container (directory) in SciDrive.

Usage

SciDrive.createContainer(path)

Arguments

path

path of the directory in SciDrive.

Value

Returns TRUE if the container (directory) was created successfully. Throws an exception if the user is not logged into SciServer (use Authentication.login for that purpose). Throws an exception if the HTTP request to the SciDrive API returns an error.
