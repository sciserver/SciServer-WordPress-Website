CasJobs.submitJob {SciServer}R Documentation

Submit asynchronous SQL query.

Description

Submits an asynchronous SQL query to the CasJobs queue.

Usage

CasJobs.submitJob(sql, context="MyDB")

Arguments

sql

sql query

context

database context

Value

Returns the CasJobs jobID (integer). Throws an exception if the user is not logged into SciServer (use Authentication.login for that purpose). Throws an exception if the HTTP request to the CasJobs API returns an error.
