SkyQuery.listAllDatasets {SciServer}R Documentation

List Datasets.

Description

Lists all available datasets (more info in http://www.voservices.net/skyquery).

Usage

SkyQuery.listAllDatasets()

Value

Returns dataset definitions as a list object. Throws an exception if the user is not logged into SciServer (use Authentication.login for that purpose). Throws an exception if the HTTP request to the SkyQuery API returns an error.
