CasJobs.executeQuery {SciServer}R Documentation

Synchronous SQL query.

Description

Executes a synchronous SQL query in a CasJobs database context.

Usage

CasJobs.executeQuery(sql, context="MyDB", format="dataframe")

Arguments

sql

sql query

context

database context

format

output format. Choose between "list" (list object), "dataframe" (data.frame object), "csv" (csv string), "json" (json string), "fits" (raw binary data containing a fits file) and "raw" (raw binary data containing a fits file).

Value

Returns the results table from the query in different output formats defined by the input parameter 'format'. Throws an exception if the HTTP request to the CasJobs API returns an error
